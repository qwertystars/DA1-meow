Build Native Code For The Web

emsdk bundles Emscripten, LLVM, Binaryen, and Node.js in one installer flow, making it easy to compile your existing C/C++ projects to fast WebAssembly output.

1) Install EMSDK

  • Clone the SDK repository.
  • Install the latest toolchain build.
  • Activate the selected version.
git clone https://github.com/emscripten-core/emsdk.git cd emsdk emsdk install latest emsdk activate latest

2) Configure Your Shell

  • Run the environment script before building.
  • Use the appropriate script per platform.
  • Verify with emcc -v.
# Windows (PowerShell) .\emsdk_env.ps1 # Linux/macOS source ./emsdk_env.sh

3) Compile C To Wasm

  • Use emcc to compile sources.
  • Generate HTML/JS/Wasm together for easy testing.
  • Run output in a local web server.
emcc main.c -O2 -s WASM=1 -o index.html
